CVIA Assignment 1 Dataset (extracted from OXFORD-IIIT PET Dataset)

-----------------------
The dataset contains 100 images of cats and dogs across 5 different breeds, 20 images for each class. These images have variations in scale, pose and lighting. 


Contents:
--------
- The "image" folder contains all the images.
- The "annotation" folder contains the associated ground truth annotation for each image (the segmentation map). Pixel Annotations: 1: Foreground 2:Background 3: Not classified.
  

References:
----------
[1] O. M. Parkhi, A. Vedaldi, A. Zisserman, C. V. Jawahar
